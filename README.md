# Body Sculpt With Sherry

Static GitHub Pages website for www.bodysculptwithsherry.com.

## How to use
1. Upload all files to a GitHub repository.
2. Go to Settings > Pages.
3. Deploy from the main branch.
4. Add your custom domain: `www.bodysculptwithsherry.com`.

## Replace placeholders
- Add Sherry's main photo in `assets/images/`.
- Update the hero image placeholder in `index.html`.
- Replace email, Instagram, and booking links in the Contact section.
